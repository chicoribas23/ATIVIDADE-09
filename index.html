Reflexão e Aplicação Prática

Propriedades CSS Utilizadas

  • display: flex - Define o container como flexível.
  • justify-content - Alinha os itens no eixo principal (horizontal).
  • align-items - Alinha os itens no eixo cruzado (vertical).
  • flex-direction - Define a direção dos elementos (linha ou coluna).

Aplicação Prática com Flexbox

Se eu fosse criar um menu de navegação responsivo, usaria Flexbox para alinhar os itens horizontalmente e, em resoluções menores, mudaria a direção para coluna. Também usaria justify-content: space-between para distribuir bem os itens e align-items: center para manter o alinhamento centralizado verticalmente.
